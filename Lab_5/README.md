# Лабораторна робота №5 👋

This is an [Expo](https://expo.dev) project created with [`create-expo-app`](https://www.npmjs.com/package/create-expo-app).

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npx expo start
   ```

## 📚 Контрольні запитання

### 1. Перенаправлення неавторизованого користувача

Реалізується через `_layout.jsx` захищеної групи маршрутів із використанням:

```
<Redirect href="/login" />
```

---

### 2. Різниця між `<Link>` та `router.push()`

| `<Link>`               | `router.push()`             |
| ---------------------- | --------------------------- |
| Декларативний          | Імперативний                |
| Використовується в JSX | Викликається у функціях     |
| Аналог `<a>`           | Аналог програмного переходу |

---

### 3. Динамічні маршрути

Створюються через `[id].jsx`.

Отримання параметрів:

```js
const { id } = useLocalSearchParams();
```

---

### 4. Чому використовують Context

* Глобальний доступ
* Уникнення prop drilling
* Централізоване управління станом

---

### 5. Групи маршрутів `(folderName)`

* Використовуються для логічного поділу
* Не впливають на URL

```
(auth)/login → /login
```
