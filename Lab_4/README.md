# Файловий Менеджер

> Мобільний застосунок для роботи з локальною файловою системою,  


---

## 🚀 Швидкий старт (3 команди)

```bash
# 1. Перейти до папки проекту
cd FileOS

# 2. Встановити залежності
npm install

# 3. Запустити
npx expo start
```


## Реалізовані функції

### 1. Навігація
- Перегляд вмісту поточної директорії (`FlatList`)
- Breadcrumb-навігація з кліком по будь-якому рівню
- Кнопка «← Назад»
- Перехід у вкладені папки
- Pull-to-refresh

### 2. Створення
- **Нова папка** — ім'я через модальне вікно
- **Новий файл** — ім'я + початковий вміст
- Анімований FAB (кнопка +) з двома підменю

### 3. Перегляд
- Відкриття текстових файлів (`.txt .md .js .ts .json .html .css .py` та ін.)
- Нумерація рядків
- Статистика: рядки / слова / символи / розмір

### 4. Редагування
- Повноцінний редактор з моноширинним шрифтом
- Індикатор незбережених змін (помаранчева крапка)
- Кнопки: Зберегти / Скасувати зміни / Вийти
- Alert при виході без збереження (зберегти / не зберігати / скасувати)

### 5.  Видалення
- Кнопка 🗑️ на кожному елементі
- Alert-підтвердження з попередженням для директорій

### 6. Деталі файлу
- Назва, тип (за розширенням), розмір, дата останньої зміни
- Повний URI-шлях
- Стан існування

### 7. Статистика пам'яті (головний екран)
- Загальний обсяг диску
- Вільний простір
- Зайнятий простір
- Візуальний кольоровий прогрес-бар
