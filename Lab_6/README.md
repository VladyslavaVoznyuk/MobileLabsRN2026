**Ключові досягнення:**
1.  Налаштовано гнучку навігацію (Expo Router) з автоматичним перенаправленням залежно від стану авторизації користувача.
2.  Вивчено роботу з **Firebase Authentication** для безпечного керування користувачами.
3.  Освоєно роботу з NoSQL базою даних **Firestore**, зокрема механізми синхронізації даних у реальному часі.
4.  Реалізовано надійну систему захисту даних за допомогою **Firestore Security Rules**, що запобігає несанкціонованому доступу до чужих профілів.
5.  Закріплено навички роботи з асинхронними запитами в React Native та обробки помилок (catch blocks) для покращення UX.
